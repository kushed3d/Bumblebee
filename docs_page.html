Documentation - All Commands

Comprehensive list of all the commands.

/ping

Gets and displays bot & command latency.

Permissions required: None.

/invite

Provides a invite for Bumblebee.

Permissions Required: None.

/verify-setup

Makes a verification system.

Permissions Required: MANAGE_GUILD.

/ticket-setup

Makes a ticket system.

Permissions Required: MANAGE_GUILD.

/say

Makes the bot say something.

Permissions Required: KICK_MEMBERS.

/rate

Rates other members.

Permissions Required: None.

/poll

Creates a poll.

Permissions Required: MANAGE_GUILD.

/kick

Kicks a user.

Permissions Required: KICK_MEMBERS.

/giveaway

Creates a giveaway.

Permissions Required: MANAGE_GUILD.

/embed

Creates a embed.

Permissions Required: MANAGE_GUILD.

/mute

Mutes a user.

Permissions Required: MUTE_MEMBERS.

/unmute

Unmutes a user.

Permissions Required: MUTE_MEMBERS.

/ban

Bans a user.

Permissions Required: BAN_MEMBERS.

/unban

Unbans a user.

Permissions Required: BAN_MEMBERS.

/purge

deletes messages.

Permissions Required: None.
